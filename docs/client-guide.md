# Guia do Cliente — Como Gerenciar Seu Site

## Acessando o Painel Administrativo

Acesse: `seusite.com.br/admin`

Use o e-mail e senha fornecidos pelo desenvolvedor.

## Editando Configurações do Site

### Informações Gerais (Site Settings)

No menu lateral, clique em **Site Settings**:

- **Nome do site**: aparece no header e título das páginas
- **Logo**: faça upload da sua logo (PNG ou SVG recomendado)
- **Favicon**: ícone que aparece na aba do navegador
- **Cores**: primária e secundária (formato hexadecimal, ex: #3b82f6)
- **Contato**: telefone, e-mail e endereço
- **Redes Sociais**: adicione links para suas redes

### Menu de Navegação (Header)

No menu lateral, clique em **Header**:

- Adicione itens ao menu clicando "Adicionar Nav Item"
- Cada item tem: texto (label) e link
- Para link interno: selecione uma página do site
- Para link externo: cole a URL completa
- Marque "Nova aba" para abrir em outra janela
- O botão CTA aparece destacado no canto direito

### Rodapé (Footer)

No menu lateral, clique em **Footer**:

- Organize links em colunas
- Edite o texto de copyright
- Habilite/desabilite a exibição de redes sociais

## Gerenciando Páginas

### Criar Nova Página

1. No menu lateral, clique em **Pages** > **Create New**
2. Preencha o título (o slug é gerado automaticamente)
3. Em **Layout**, adicione blocos:
   - **Hero**: banner principal com imagem de fundo
   - **Content**: texto rico com formatação
   - **Features**: grade de serviços ou recursos
   - **CTA**: chamada para ação
   - **Testimonials**: depoimentos de clientes
   - **FAQ**: perguntas frequentes
   - **Gallery**: galeria de imagens
   - **Pricing**: tabela de preços
   - **Banner**: banner promocional
4. Configure o SEO na aba **SEO**
5. Mude o status para **Publicado**
6. Clique **Salvar**

### Editar Página Existente

1. Clique em **Pages** no menu lateral
2. Clique na página desejada
3. Faça as alterações
4. Clique **Salvar**

## Gerenciando Blog

### Criar Post

1. Clique em **Posts** > **Create New**
2. Preencha título, conteúdo e imagem destacada
3. Selecione categorias
4. Configure SEO
5. Publique

### Gerenciar Categorias

Em **Categories**, crie e organize as categorias do blog.

## Upload de Mídia

1. Clique em **Media** no menu lateral
2. Faça upload de imagens
3. **IMPORTANTE**: sempre preencha o campo "Texto alternativo" (alt) para acessibilidade

## Configurações SEO

No menu lateral, clique em **SEO Settings**:

- **Google Analytics ID**: formato G-XXXXXXXXXX
- **Google Search Console**: código de verificação
- **Google Tag Manager**: formato GTM-XXXXXXX
- **Imagem OG padrão**: imagem usada quando a página não tem uma específica

## Configurações de Marketing

No menu lateral, clique em **Marketing Settings**:

- **Meta Pixel**: ID do pixel do Facebook/Instagram
- **Google Ads**: ID de conversão
- **TikTok Pixel**: ID do pixel
- Habilite/desabilite cada pixel individualmente
- **Scripts customizados**: para qualquer outro código de rastreamento

## Dicas

- Sempre salve após fazer alterações
- Use o botão "Preview" para visualizar antes de publicar
- Imagens grandes são redimensionadas automaticamente
- O sitemap é atualizado automaticamente ao publicar/despublicar páginas
