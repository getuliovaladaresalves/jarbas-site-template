# Checklist — Configuração SEO

## Google Search Console
- [ ] Verificar propriedade do site
- [ ] Submeter sitemap.xml
- [ ] Verificar cobertura de índice
- [ ] Configurar alertas de erros

## Google Analytics 4
- [ ] Criar propriedade GA4
- [ ] Adicionar Measurement ID no admin (SEO Settings)
- [ ] Verificar rastreamento de page views
- [ ] Configurar eventos de conversão

## Meta Tags (por página)
- [ ] Title tag único e descritivo (50-60 chars)
- [ ] Meta description (150-160 chars)
- [ ] Open Graph title e description
- [ ] Open Graph image (1200x630px recomendado)
- [ ] Canonical URL definido

## Estrutura
- [ ] sitemap.xml gerado e acessível
- [ ] robots.txt configurado
- [ ] Redirects 301 para URLs antigas
- [ ] URLs amigáveis (slugs limpos)
- [ ] Breadcrumbs implementados

## Schema.org
- [ ] Organization markup
- [ ] WebPage markup nas páginas
- [ ] Article markup nos posts
- [ ] BreadcrumbList markup
- [ ] LocalBusiness (se aplicável)

## Técnico
- [ ] HTTPS em todas as páginas
- [ ] Sem conteúdo duplicado
- [ ] Sem links quebrados (404)
- [ ] Core Web Vitals satisfatórios
- [ ] Mobile-friendly (responsive)
