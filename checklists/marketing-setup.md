# Checklist — Configuração de Marketing

## Meta Pixel (Facebook/Instagram)
- [ ] Criar pixel no Meta Business Suite
- [ ] Adicionar ID no admin (Marketing Settings)
- [ ] Habilitar o pixel
- [ ] Verificar PageView no Events Manager
- [ ] Configurar eventos de conversão (Lead, Purchase, etc.)
- [ ] Testar com Facebook Pixel Helper (extensão Chrome)

## Google Ads
- [ ] Criar conta Google Ads
- [ ] Obter Conversion ID (AW-XXXXXXXXX)
- [ ] Obter Conversion Label
- [ ] Adicionar IDs no admin
- [ ] Verificar tag no Google Tag Assistant
- [ ] Configurar remarketing audience

## Google Tag Manager
- [ ] Criar container GTM
- [ ] Adicionar GTM ID no admin (SEO Settings)
- [ ] Configurar tags, triggers e variáveis
- [ ] Publicar container

## TikTok Pixel (se aplicável)
- [ ] Criar pixel no TikTok Ads Manager
- [ ] Adicionar ID no admin
- [ ] Verificar PageView

## LGPD / Consent
- [ ] ConsentBanner aparece na primeira visita
- [ ] Scripts de rastreamento só carregam após aceite
- [ ] Botão "Recusar" funciona (scripts não carregam)
- [ ] Link para Política de Privacidade funciona
- [ ] Preferência salva no localStorage

## UTM & Tracking
- [ ] URLs de campanha com UTM parameters
- [ ] Conversões rastreando corretamente
- [ ] Dashboard de analytics acessível
